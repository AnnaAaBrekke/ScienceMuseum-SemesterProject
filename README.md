# "Futures Museum" Website Project (Community Science Museum)

## Overview
Creating a modern, accessible, and responsive web presence for the Community Science Museum ("Futures Museum"), an interactive science museum aimed at primary and middle school children (ages 7-15) and families with young children.

## Description
The Community Science Museum ("Futures Museum") project involves designing and building a website to engage and entice visitors. The site will use provided images and text content to showcase exhibits, events, and visitor information in a manner that appeals to youngsters without being overtly childish.

## Features
- **Interactive Exhibit Showcase**: A dynamic page displaying various exhibits with interactive elements.
- **Detailed Exhibit Pages**: Pages for each exhibit with in-depth information, images, and interactive features.
- **Visitor Information**: Detailed information for visitors, including hours of operation, ticket prices, and directions.
- **Contact Page**: A contact form and information for visitors to get in touch with the museum staff.
- **Placeholder Logo**: A custom placeholder logo to represent the museum until the final design is decided by the committee.
- **Responsive Design**: Ensures the website is accessible and looks great on all devices, including desktops, tablets, and mobile phones.
- **Accessible Design**: Follows best practices for accessibility to ensure the site is usable by all visitors, including those with disabilities.
